# cxcli tts synthesize

Transforms text into audio

```
cxcli tts synthesize [input] [flags]
```

## Options

```
  -h, --help            help for synthesize
  -l, --locale string   Input locale
  -o, --output string   Output file name (default "output.mp3")
```
